Teem - Chat App

Full Stack Developer


Designed & built a Chat web App using MERN Stack where users can able to chat on one to one and Team chat as well.

User based authentication is implemented by using JWT. Seperate UI was created where users can signin and signup. Chakra UI is implemented for responsive & attractive User Interface. Real time Chat access is developed by socket io.

Group chat with more then 3 users is possible.Lets Chat... Note: Get Guest user Credentials by click on the button in the Login page.

Note: Users can access their profile, create group chat, update group name, add and remove friends on their group chat.
