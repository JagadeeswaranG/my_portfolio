Paytm - Clone App

Full Stack Developer


Designed & built a Paytm Clone web App using MERN Stack where users can able to pay their bills on different categories of service payments & get E-Invoice to the registered Email Id. All services in One App and so payments are simple now.

User based authentication is implemented by using JWT. Seperate UI was created where user can register, signin, signup and reset the password by using forgotpassword. Nodemailer is used to get the e-Invoice with payment details once the user submit the order.

Sign Up with simple steps... Get your signin Credentials. Note: Please registered with valid Email Id Where You can get your E-Invoice.
