Online Class Ticketing system for query resolving

Full Stack Developer


Designed & developed an online students portal for query resolving using MERN Stack where the students are able to raise their queries and get resolved by the suitable mentors. This dedicated portal would reduces the students offline engagement and improves their learning capabilities.

Role based authentication is implemented so that admin, mentors and students can be able to login for their portal access through the same UI page.All the query status have been maintained by Admin portal and every users can access for their status history in their respective portals.
